# Baseball Stats

Prerequisites:

* Ruby version - 2.5.3
* Rails - 5.2.4

##### Gem installation:
$ bundle install

##### DB Creation:
$ rails db:create

##### DB Migration:
$ rails db:migrate

##### Load sample Data:
$ rails db:seed

##### Above Commands together:
$ rails db:reset

##### To Start Server:
$ rails s

##### To Start Console:
$ rails c
